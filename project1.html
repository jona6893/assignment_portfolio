The Project

This project involved the use of fonts and how to incorporate them bast into the design of a website. Making sure that the readability of the site stay intact. This was done by first picking one or two fonts. I decide to pick one font to work with. That font being “Quicksand”. The next step was to make sure that the text was readable by checking the line-height and the font-weight depending on whether it’s a headline or a paragraph. On the card layout a used CSS grid with an auto fit property that would increase along with the width of the website, making it usable on both desktop and mobile.
