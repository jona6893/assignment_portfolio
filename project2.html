Color Palette

The Project

In this project we had to take an existing website template, and style the site using a custom color platte, animations and a Darkmode. Using huemint.com I was able to find a palette to my liking, as well as colours which would work with high contrast. In order to check the readability of the color I used Adobe Color Analyser. Using this a could be sure that the palette contrast was readable. With darkmode I add a toggle to the navbar, which when clicked would toggle list of css classes which followed an alternative colorplatte. Although this can be done using data-attribute I decided not to use that route. On the rocket ship image a added an animation which would trigger when hoved over.
