In order the better make the website responsive the use of media queries allow for specific CSS code at specific Viewport Width.

CSS flex and Grid

Flex and Grid are the building blocks of a websites layout. Utilising them we are able to make the content more flex and responsive to the devices which are used.

The Project

The project involved take a static website and turning it into one which can be used of a wide verity of devices. In order to do this, I had to make sure that the section of the website used ether CSS Flex or CSS grid depending on the use case. Flex is a more responsive layout but does not work all layouts. Grid on the other hand is not a way responsive layout but can be when use the “auto-fit(min-max)” or “auto-fit(min-max)”. To make the text on the site responsive, I decided to use media queries instead of font clamps. This also allowed me to make change to element where as the font clamp would only allow for changes to the font.
